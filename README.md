# 코레일 기차표 예매

## 목표
* 코레일 기차표 자동 예매

## 차별점
* 텔레그램과 연동
    - 텔레그램을 통해 예매 여부 알림 ( 결제는 사용자가 최종 확인 후 진행 - 수동 )
    - 텔레그램 채팅 내용을 바탕으로 크롤링 진행 ( 따로 크롤링 코드 실행 필요없이 )


## 진행상황
1. 예매 진행
2. 텔레그램을 이용하여 예매 완료 메시지 송신
3. 텔레그램 채팅을 이용하여 원하는 기차 선택 & 시작 가능



## 폴더 구조
- crawling
    - 코레일 데이터 정보를 크롤링 하는 코드
- telegram
    - 텔레그램 서비스 관련 폴더 
-sLLM
    - sLLM 추론 관련 폴더 
- main
    - 실제 서비스가 실행되는 폴더
